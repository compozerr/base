# Create a project

A **project** is a full-stack application that can be shared with the community. It can include multiple modules, each with its own frontend and backend implementations.

## How to

To create a new project, use the following command:

```bash
compozerr new project <name>
```

This command will generate a template project in your `projects` folder, providing you with a starting point for your development.

:::warning Requires login
This command requires you to be [logged in](./login.md) to your compozerr account.
:::
