# Login

To use the `compozerr` CLI, you need to be logged in to your compozerr account. This allows you to deploy your projects and modules, as well as access other features.

## How to

To log in to your compozerr account, use the following command:

```bash
compozerr login
```

This command will open your default browser and wait for you to login with GitHub. Once you've logged in, you can close the browser window and return to the terminal.
